# Input Library

## Methods

### > IsButtonDown( btn: **int** )
returns **bool**

### > IsButtonPressed( btn: **int** )
returns **pressed: bool**, tick: **int**

The **tick** return value is the tick when it was pressed

### > IsButtonReleased( btn: **int** )
returns **pressed: bool**, tick: **int**

The **tick** return value is the tick when it was released
