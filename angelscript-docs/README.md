# README

Welcome to the docs :)

If you don't know AngelScript, you can find more about it in the [official website](https://angelcode.com/angelscript/)

You can get autocompletion & intellisense by placing `as.predefined` in the same folder as your other Angelscript scripts

Here is a template you can follow to start making Angelscripts for Skill Issue:

```as
// called when
// loading the script
void main()
{
	// Code goes here
}

// called when
// unloading the script
void unload()
{
	// Code goes here
}
```
